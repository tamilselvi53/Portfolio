Sorting Visualizer

Overview

It is a tool for visualizing sorting algorithms. Different colors help in understanding the sorting process. Blue is the default color of all elements. Yellow represents the current position of the array. Red represents swapping or change in position. Green represents that element that found its position. Time and space complexity of the algorithm is displayed. Features: Speed of the algorithm Size of the array Generate new array Bubble Insertion Selection Merge Quick Heap

Problem

It is quite difficult to understand Sorting Algorithms for beginners. Without having good foundation leads to lack in problem solving. Help them by building a tool which describes implementation of the algorithm in simple way.

Solution

Visualizing a complex algorithm makes it easy to understand. I have developed a website which implements sorting algorithm with color variations.
